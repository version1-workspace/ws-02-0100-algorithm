# Problem Creator

Problem Creatorは、アルゴリズムとデータ構造の問題を作成するためのスキルです。
またそれを確認するための
- テストケース
- 模範回答
も用意します。

## このスキルを使う場面

- アルゴリズムとデータ構造の問題を作成したいとき
- 問題のテストケースを用意したいとき

# 作問のステップ

1. 問題のタイトルとカテゴリを確認する
2. タイトルとカテゴリに基づいて問題文を作成する
3. 問題文に条件や例を追加する
4. 問題文に入力制約を追加する
5. 問題文を確認する
6. 問題文に基づいてテストケースを用意する
7. テストケースを確認する
8. 問題文とテストケースの整合性を確認する
9. 模範回答を作成する


# 問題テンプレート

```
---
title: タイトル
category: 問題カテゴリ
---

# 問題文

整数の配列 nums と整数 target が与えられます。
nums の中から 合計が target になる2つの要素のインデックス を返してください。

# 条件

- 各入力には 必ず1つだけ解が存在する と仮定してよい
- 同じ要素を2回使ってはいけない
- 答えの順番はどちらでもよい


# 例

### 例1

入力: nums = [2,7,11,15], target = 9
出力: [0,1]
説明: target = nums[0] + nums[1] となるため、[0, 1] を返します。

### 例2

入力: nums = [3,2,4], target = 6
出力: [1,2]

### 例3
入力: nums = [3,3], target = 6
出力: [0,1]


# 入力制約

- 2 <= nums.length <= 10^4
- -10^9 <= nums[i] <= 10^9
- -10^9 <= target <= 10^9

```


# 入力

- タイトル
- 問題カテゴリ


# 出力

- 問題文(制約条件や入力例なども含む)
- テストケース
- 模範回答

出力方式はプロジェクトのドキュメントに従う。
