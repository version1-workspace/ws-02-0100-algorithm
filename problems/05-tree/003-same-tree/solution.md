# 解説

## 考え方

2つの木が同じであるためには、現在見ているノードの有無、値、左部分木、右部分木がすべて一致している必要があります。この条件を各ノードで再帰的に確認します。

## アルゴリズム

1. `p` または `q` のどちらかが `null` なら、両方が `null` かを返す
2. 両方ともノードなら、値 `p.val` と `q.val` を比較する
3. 左部分木同士を再帰的に比較する
4. 右部分木同士を再帰的に比較する
5. 値、左、右がすべて一致すれば `true` を返す

## 実装のポイント

- `if (p === null || q === null) return p === q;` により、両方空なら `true`、片方だけ空なら `false` になります。
- 値だけでなく、左右の構造も `isSameTree` で確認しています。
- `&&` でつないでいるため、どこかで不一致があればそれ以上の比較は不要です。

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(h)

ただし `n` は比較するノード数、`h` は木の高さです。
