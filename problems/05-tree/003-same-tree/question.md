# 問題文

2つの二分木の根ノード `p` と `q` が与えられます。
2つの木が同じ構造を持ち、対応するすべてのノードの値も同じなら `true` を返してください。
そうでなければ `false` を返してください。

各ノードは次の `TreeNode` で表されます。

```ts
class TreeNode {
  val: number;
  left: TreeNode | null;
  right: TreeNode | null;
}
```

# 条件

- 2つの木がどちらも空なら同じ木とみなす
- 片方だけが空なら同じ木ではない
- ノードの値だけでなく左右の子の構造も一致している必要がある

# 例

### 例1

入力: `p = [1, 2, 3], q = [1, 2, 3]`
出力: `true`

### 例2

入力: `p = [1, 2], q = [1, null, 2]`
出力: `false`

### 例3

入力: `p = [1, 2, 1], q = [1, 1, 2]`
出力: `false`

# 入力制約

- 各木のノード数は `0` 以上 `100` 以下
- `-10^4 <= Node.val <= 10^4`
