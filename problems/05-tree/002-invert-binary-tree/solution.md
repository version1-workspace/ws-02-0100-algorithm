# 解説

## 考え方

木全体を反転するには、すべてのノードで左部分木と右部分木を入れ替えれば十分です。部分木も同じ操作で反転できるため、再帰で処理します。

## アルゴリズム

1. `root` が `null` なら `null` を返す
2. 左部分木を再帰的に反転する
3. 右部分木を再帰的に反転する
4. 反転済みの右部分木を `root.left` に、反転済みの左部分木を `root.right` に代入する
5. `root` を返す

## 実装のポイント

- `left` と `right` に再帰結果を一度保存してから入れ替えています。
- 新しい木を作るのではなく、渡されたノードの `left` と `right` を直接更新します。
- 空の木では何も入れ替えるものがないため `null` を返します。

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(h)

ただし `n` はノード数、`h` は木の高さです。
