# 問題文

二分木の根ノード `root` が与えられます。
この木が有効な二分探索木なら `true` を返し、そうでなければ `false` を返してください。

有効な二分探索木では、すべてのノードについて次の条件を満たします。

- 左部分木に含まれるすべての値は、そのノードの値より小さい
- 右部分木に含まれるすべての値は、そのノードの値より大きい
- 左右の部分木もそれぞれ有効な二分探索木である

各ノードは次の `TreeNode` で表されます。

```ts
class TreeNode {
  val: number;
  left: TreeNode | null;
  right: TreeNode | null;
}
```

# 条件

- 重複する値は二分探索木として無効とする
- 直下の子だけでなく、部分木全体が親から受け継ぐ範囲制約を満たす必要がある
- 空の木は有効な二分探索木とみなす

# 例

### 例1

入力: `root = [2, 1, 3]`
出力: `true`

### 例2

入力: `root = [5, 1, 4, null, null, 3, 6]`
出力: `false`

### 例3

入力: `root = [2, 2, 3]`
出力: `false`

# 入力制約

- ノード数は `0` 以上 `10^4` 以下
- `-2^31 <= Node.val <= 2^31 - 1`
