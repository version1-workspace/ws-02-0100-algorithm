# 問題文

二分木の根ノード `root` が与えられます。
この二分木の直径を返してください。

二分木の直径とは、任意の2つのノード間を結ぶ最長経路に含まれる辺の数です。
この経路は根ノードを通るとは限りません。

各ノードは次の `TreeNode` で表されます。

```ts
class TreeNode {
  val: number;
  left: TreeNode | null;
  right: TreeNode | null;
}
```

# 条件

- 返す値はノード数ではなく辺の数とする
- 空の木またはノードが1つだけの木の直径は `0` とする
- 各ノードで左右の部分木の高さを使って最長経路を考える

# 例

### 例1

入力: `root = [1, 2, 3, 4, 5]`
出力: `3`
説明: 最長経路は `4 -> 2 -> 1 -> 3` または `5 -> 2 -> 1 -> 3` で、辺の数は `3` です。

### 例2

入力: `root = [1, 2]`
出力: `1`

### 例3

入力: `root = []`
出力: `0`

# 入力制約

- ノード数は `0` 以上 `10^4` 以下
- `-100 <= Node.val <= 100`
