# 解説

## 考え方

二分木の最大深さは、左部分木の最大深さと右部分木の最大深さの大きい方に、現在のノード1つ分を足したものです。空の木は深さ `0` として扱います。

## アルゴリズム

1. `root` が `null` なら `0` を返す
2. 左部分木の深さを再帰的に求める
3. 右部分木の深さを再帰的に求める
4. 大きい方に `1` を足して返す

## 実装のポイント

- `maxDepth(root.left)` と `maxDepth(root.right)` が、それぞれ左右の最大深さを返します。
- `Math.max(...)+1` によって、現在のノードを含めた深さにしています。
- 根だけの木では左右がどちらも `0` なので、結果は `1` になります。

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(h)

ただし `n` はノード数、`h` は木の高さです。再帰呼び出しのスタック分だけ空間を使います。
