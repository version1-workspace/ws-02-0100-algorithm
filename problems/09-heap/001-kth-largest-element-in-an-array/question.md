# 問題文

整数配列 `nums` と整数 `k` が与えられます。
配列の中で `k` 番目に大きい要素を返してください。

ここで `k` 番目に大きい要素は、ソート後の重複を含む順位で数えます。

# 条件

- 重複する値も別々の要素として数える
- 配列全体を昇順に並べたとき、後ろから `k` 番目の値を返す
- Heap または Quickselect を使うと効率よく求められる

# 例

### 例1

入力: `nums = [3, 2, 1, 5, 6, 4]`, `k = 2`
出力: `5`

### 例2

入力: `nums = [3, 2, 3, 1, 2, 4, 5, 5, 6]`, `k = 4`
出力: `4`

# 入力制約

- `1 <= k <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
