# 問題文

整数配列 `nums` と整数 `k` が与えられます。
出現回数が多い上位 `k` 個の要素を返してください。

runner で比較しやすいよう、返り値は出現回数の降順で並べ、出現回数が同じ場合は値の昇順で返してください。

# 条件

- 答えは必ず一意に定まるとは限らないため、この問題では並び順を明確に指定する
- Heap または Bucket を使うと効率よく求められる
- `k` は異なる値の個数以下で与えられる

# 例

### 例1

入力: `nums = [1, 1, 1, 2, 2, 3]`, `k = 2`
出力: `[1, 2]`

### 例2

入力: `nums = [1]`, `k = 1`
出力: `[1]`

# 入力制約

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
- `1 <= k <= 異なる値の個数`
