# 問題文

昇順にソートされた `k` 個の単方向連結リスト `lists` が与えられます。
すべてのリストを1つの昇順リストにマージし、マージ後の先頭ノードを返してください。

各ノードは次の `ListNode` で表されます。

```ts
class ListNode {
  val: number;
  next: ListNode | null;
}
```

# 条件

- 新しい値の配列を返すのではなく、マージ後の `ListNode | null` を返す
- 空のリストが含まれる場合もある
- Min Heap を使うと、各リストの先頭候補から最小値を効率よく選べる

# 例

### 例1

入力: `lists = [[1, 4, 5], [1, 3, 4], [2, 6]]`
出力: `[1, 1, 2, 3, 4, 4, 5, 6]`

### 例2

入力: `lists = []`
出力: `[]`

### 例3

入力: `lists = [[]]`
出力: `[]`

# 入力制約

- `0 <= lists.length <= 10^4`
- 各リストのノード数の合計は `0` 以上 `10^4` 以下
- `-10^4 <= Node.val <= 10^4`
- 各リストは昇順にソートされている
