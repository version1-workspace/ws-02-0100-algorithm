# 問題文

CPU が実行するタスク列 `tasks` と、同じ種類のタスクを再実行するまでに必要な待機時間 `n` が与えられます。
各タスクは1単位時間で実行できます。

同じ種類のタスクの間には、少なくとも `n` 単位時間の間隔を空ける必要があります。
すべてのタスクを完了するために必要な最短時間を返してください。

# 条件

- 待機時間中は idle として時間を消費する
- 異なる種類のタスクは連続して実行できる
- 出現回数の多いタスクを優先する Greedy や Heap で考えられる

# 例

### 例1

入力: `tasks = ["A", "A", "A", "B", "B", "B"]`, `n = 2`
出力: `8`

説明: `A -> B -> idle -> A -> B -> idle -> A -> B` のように実行できます。

### 例2

入力: `tasks = ["A", "C", "A", "B", "D", "B"]`, `n = 1`
出力: `6`

### 例3

入力: `tasks = ["A", "A", "A", "B", "B", "B"]`, `n = 3`
出力: `10`

# 入力制約

- `1 <= tasks.length <= 10^4`
- `tasks[i]` は大文字英字
- `0 <= n <= 100`
