# 解説

## 考え方

各位置の答えは「左側すべての積」と「右側すべての積」の積です。割り算を使わず、左から prefix、右から suffix を掛け合わせます。

## アルゴリズム

1. `result` をすべて `1` で初期化する
2. 左から走査し、`result[i]` に `i` より左側の積 `prefix` を入れる
3. `prefix` に `nums[i]` を掛けて次へ進む
4. 右から走査し、`result[i]` に `i` より右側の積 `suffix` を掛ける
5. `suffix` に `nums[i]` を掛けて次へ進み、最後に `result` を返す

## 実装のポイント

- `prefix` と `suffix` は、現在位置の値を含まない積として使う
- `0` が含まれていても、積をそのまま伝播させるため特別扱いは不要
- 出力配列 `result` を使って左側の積を保持し、右側の積を後から掛けている

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(1) 追加領域。ただし返却する `result` は O(n)
