# 問題文

文字列 `s` と `t` が与えられます。
`t` に含まれるすべての文字を、出現回数も含めて含む `s` の最短の連続部分文字列を返してください。
条件を満たす部分文字列が存在しない場合は空文字列 `""` を返してください。

# 条件

- `t` に同じ文字が複数回出現する場合、その回数以上含める必要がある
- 返す文字列は `s` の連続部分文字列でなければならない
- 最短の部分文字列が複数ある場合は、そのうち 1 つを返せばよい

# 例

### 例1

入力: `s = "ADOBECODEBANC"`, `t = "ABC"`
出力: `"BANC"`

### 例2

入力: `s = "a"`, `t = "a"`
出力: `"a"`

### 例3

入力: `s = "a"`, `t = "aa"`
出力: `""`

# 入力制約

- `1 <= s.length, t.length <= 10^5`
- `s` と `t` は英字で構成される
