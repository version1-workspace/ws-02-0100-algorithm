# 問題文

2 つの文字列 `s` と `t` が与えられます。
`t` が `s` のアナグラムであれば `true`、そうでなければ `false` を返してください。

アナグラムとは、文字の出現回数を保ったまま並び替えて作れる文字列です。

# 条件

- 文字の種類だけでなく出現回数も一致している必要がある
- 文字列の長さが異なる場合はアナグラムではない
- 文字の順番は一致していなくてもよい

# 例

### 例1

入力: `s = "anagram"`, `t = "nagaram"`
出力: `true`

### 例2

入力: `s = "rat"`, `t = "car"`
出力: `false`

# 入力制約

- `0 <= s.length, t.length <= 5 * 10^4`
- `s` と `t` は英小文字で構成される
