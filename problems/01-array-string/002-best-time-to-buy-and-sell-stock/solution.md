# 解説

## 考え方

売る日を固定して考えると、その日までに最も安く買えていれば利益が最大になります。配列を左から見ながら、これまでの最小価格と最大利益を更新します。

## アルゴリズム

1. `minPrice` を無限大、`maxProfit` を `0` で初期化する
2. 各 `price` について、`minPrice` をこれまでの最小値に更新する
3. `price - minPrice` をその日に売った場合の利益として計算する
4. `maxProfit` を最大の利益に更新する
5. 最後に `maxProfit` を返す

## 実装のポイント

- `minPrice` は現在日以前の最小価格を表す
- 利益が出ない場合は取引しない選択ができるため、`maxProfit` は `0` から始める
- 1 回の走査だけで、買う日が売る日より前という条件を満たせる

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(1)
