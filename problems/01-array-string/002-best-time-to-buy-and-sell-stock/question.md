---
title: Best Time to Buy and Sell Stock
category: Array / Greedy
---

# 問題文

整数配列 `prices` が与えられます。
`prices[i]` は `i` 日目の株価を表します。

1 回だけ株を買い、その後の別の日に 1 回だけ売ることで得られる最大利益を返してください。
利益を得られない場合は `0` を返してください。

# 条件

- 買う日は売る日より前でなければならない
- 売買は最大 1 回だけ行える
- 株を買わない選択もできる

# 例

### 例1

入力: `prices = [7, 1, 5, 3, 6, 4]`
出力: `5`
説明: 価格 `1` の日に買い、価格 `6` の日に売ると利益は `5` です。

### 例2

入力: `prices = [7, 6, 4, 3, 1]`
出力: `0`
説明: 利益を得られる売買がないため `0` を返します。

# 入力制約

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`
