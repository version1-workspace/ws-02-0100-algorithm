# 解説

## 考え方

重複のない連続部分文字列を、左右の境界 `left` と `right` で表します。文字が前に出ていて、かつ現在の窓の中にある場合だけ、`left` をその次の位置まで進めます。

## アルゴリズム

1. 各文字の最後に出た位置を `lastSeen` に保存する
2. `left` を現在の窓の左端、`best` を最長長さとして初期化する
3. `right` を左から右へ動かす
4. 現在文字が窓内で重複するなら、`left` を前回位置の次へ進める
5. 現在文字の位置を更新し、`right - left + 1` で `best` を更新する

## 実装のポイント

- `previousIndex >= left` の条件により、すでに窓の外に出た同じ文字は無視する
- `lastSeen` には文字ごとの最新インデックスだけを持つ
- 空文字列ではループが実行されず、初期値 `best = 0` が返る

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(k)。`k` は文字の種類数
