title: Maximum Subarray
category: Array / Dynamic Programming

# 問題文

整数配列 `nums` が与えられます。
連続する 1 個以上の要素からなる部分配列のうち、合計値が最大となるものの合計を返してください。

# 条件

- 部分配列は元の配列内で連続していなければならない
- 少なくとも 1 個の要素を選ぶ
- 配列に負の数だけが含まれる場合もある

# 例

### 例1

入力: `nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]`
出力: `6`
説明: 部分配列 `[4, -1, 2, 1]` の合計が最大で `6` です。

### 例2

入力: `nums = [1]`
出力: `1`

### 例3

入力: `nums = [5, 4, -1, 7, 8]`
出力: `23`

# 入力制約

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
