# 解説

## 考え方

各位置で終わる最大部分配列和を考えます。直前までの和に現在値を足すより、現在値から始め直した方が大きい場合は、そこで部分配列をリセットします。

## アルゴリズム

1. `current` と `best` を `nums[0]` で初期化する
2. 2 要素目以降を順に見る
3. `current` を「現在値だけ」または「前の `current` に現在値を足す」の大きい方に更新する
4. `best` を `current` との最大値に更新する
5. 最後に `best` を返す

## 実装のポイント

- `current` は「今の位置で終わる最大部分配列和」を表す
- `best` はこれまで見つかった全体の最大値を表す
- 全要素が負の場合でも、`nums[0]` から始めることで最大の負数を正しく返せる

## なぜ正しいか

位置 `i` で終わる部分配列は、`nums[i]` だけから始めるか、位置 `i - 1` で終わる最良の部分配列に `nums[i]` を追加するかのどちらかです。各位置でこの最良値を更新し、その最大値を `best` に残すため、全体の最大部分配列和が得られます。

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(1)
