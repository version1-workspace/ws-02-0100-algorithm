# 問題文

整数配列 `nums` が与えられます。
和が `0` になる 3 つの異なる要素の組み合わせをすべて返してください。

# 条件

- 返す各組み合わせは `[a, b, c]` の形にする
- `a + b + c = 0` を満たす必要がある
- 同じインデックスの要素を 2 回以上使ってはいけない
- 同じ値の組み合わせを重複して返してはいけない
- 組み合わせの順序、および返す配列全体の順序は問わない

# 例

### 例1

入力: `nums = [-1, 0, 1, 2, -1, -4]`
出力: `[[-1, -1, 2], [-1, 0, 1]]`

### 例2

入力: `nums = [0, 1, 1]`
出力: `[]`

### 例3

入力: `nums = [0, 0, 0]`
出力: `[[0, 0, 0]]`

# 入力制約

- `0 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`
