# 問題文

整数配列 `nums` が与えられます。
配列内に同じ値が 2 回以上出現する場合は `true`、すべての値が一意であれば `false` を返してください。

# 条件

- 値が重複しているかどうかだけを判定する
- 重複している値や位置を返す必要はない
- 空配列または要素数 1 の配列には重複は存在しない

# 例

### 例1

入力: `nums = [1, 2, 3, 1]`
出力: `true`

### 例2

入力: `nums = [1, 2, 3, 4]`
出力: `false`

### 例3

入力: `nums = [1, 1, 1, 3, 3, 4, 3, 2, 4, 2]`
出力: `true`

# 入力制約

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
