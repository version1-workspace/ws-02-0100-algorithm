# 解説

## 考え方

同じ値がすでに出現したかだけを知ればよいので、見た値を `Set` に保存します。新しい値を読むたびに `Set` に存在するか確認し、存在すれば重複です。

## アルゴリズム

1. 空の `seen` を用意する
2. `nums` を左から順に見る
3. `seen` に現在値があれば `true` を返す
4. なければ現在値を `seen` に追加する
5. 最後まで重複がなければ `false` を返す

## 実装のポイント

- `Set<number>` は値の存在確認を平均 O(1) で行える
- 重複を見つけた時点で早期 return する
- 空配列や要素数 1 の配列ではループ中に重複が見つからず、自然に `false` になる

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(n)
