
# 問題

整数の配列 nums と整数 target が与えられます。
nums の中から 合計が target になる2つの要素のインデックス を返してください。

# 条件

- 各入力には 必ず1つだけ解が存在する と仮定してよい
- 同じ要素を2回使ってはいけない
- 答えの順番はどちらでもよい

# 例

```
入力: nums = [2,7,11,15], target = 9
出力: [0,1]
説明: target = nums[0] + nums[1] となるため、[0, 1] を返します。
```

```
入力: nums = [3,2,4], target = 6
出力: [1,2]
```

```
入力: nums = [3,3], target = 6
出力: [0,1]
```

# 入力制約

- 2 <= nums.length <= 10^4
- -10^9 <= nums[i] <= 10^9
- -10^9 <= target <= 10^9
