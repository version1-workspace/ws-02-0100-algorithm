# Valid Parentheses

## 問題

文字列 `s` が与えられます。
この文字列は次の3種類の文字だけで構成されています：

- `'('`, `')'`
- `'{'`, `'}'`
- `'['`, `']'`

このとき、文字列が **有効（valid）** かどうかを判定してください。

---

## 有効な条件

以下のすべてを満たす必要があります：

1. 開き括弧は、同じ種類の閉じ括弧で閉じられている
2. 開き括弧は、正しい順序で閉じられている
3. すべての閉じ括弧には、対応する開き括弧が存在する

---

## 例

### 例1

```
入力: s = "()"
出力: true
```


### 例2

```
入力: s = "()[]{}"
出力: true
```

### 例3

```
入力: s = "(]"
出力: false
```
