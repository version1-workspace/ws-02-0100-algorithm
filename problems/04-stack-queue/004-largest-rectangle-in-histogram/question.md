# Largest Rectangle in Histogram

## 問題

ヒストグラムの各棒の高さを表す整数配列 `heights` が与えられます。
各棒の幅はすべて `1` です。

隣り合う棒をいくつか選んで作れる長方形のうち、最大の面積を返してください。

---

## 条件

- 長方形は連続した範囲の棒だけを使って作ります
- 長方形の高さは、選んだ範囲内の最小の棒の高さになります
- 単調増加スタックを使うと、各棒を高々1回ずつ push / pop して解けます

---

## 例

### 例1

```
入力: heights = [2,1,5,6,2,3]
出力: 10
```

説明: 高さ `5` と `6` の2本を使うと、面積 `5 * 2 = 10` の長方形を作れます。

### 例2

```
入力: heights = [2,4]
出力: 4
```

### 例3

```
入力: heights = [1,1,1]
出力: 3
```

---

## 入力制約

- `1 <= heights.length <= 10^5`
- `0 <= heights[i] <= 10^4`
