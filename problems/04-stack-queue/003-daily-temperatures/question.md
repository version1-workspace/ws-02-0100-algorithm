# Daily Temperatures

## 問題

日ごとの気温を表す整数配列 `temperatures` が与えられます。
各日について、より高い気温の日が来るまで何日待つ必要があるかを返してください。

より高い気温の日が存在しない場合、その日の答えは `0` にしてください。

---

## 条件

- 未来の日だけを見て判定します
- 同じ気温は「より高い気温」には含みません
- 単調スタックを使うと、各要素を高々1回ずつ push / pop して解けます

---

## 例

### 例1

```
入力: temperatures = [73,74,75,71,69,72,76,73]
出力: [1,1,4,2,1,1,0,0]
```

### 例2

```
入力: temperatures = [30,40,50,60]
出力: [1,1,1,0]
```

### 例3

```
入力: temperatures = [30,60,90]
出力: [1,1,0]
```

---

## 入力制約

- `1 <= temperatures.length <= 10^5`
- `30 <= temperatures[i] <= 100`
