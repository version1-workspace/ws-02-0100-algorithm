# 問題文

階段を `n` 段のぼります。
1 回の移動で `1` 段または `2` 段のぼることができます。

ちょうど `n` 段目に到達する方法は何通りあるかを返してください。

# 条件

- 各移動では `1` 段または `2` 段だけ進める
- 移動の順番が異なる場合は別の方法として数える
- `n = 1` の場合は `1` を返す

# 例

### 例1

入力: `n = 2`
出力: `2`
説明: `1 + 1`、`2` の 2 通りです。

### 例2

入力: `n = 3`
出力: `3`
説明: `1 + 1 + 1`、`1 + 2`、`2 + 1` の 3 通りです。

# 入力制約

- `1 <= n <= 45`
