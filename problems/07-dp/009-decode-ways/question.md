# 問題文

数字だけで構成された文字列 `s` が与えられます。
`"1"` から `"26"` はそれぞれ `"A"` から `"Z"` に対応しています。

文字列 `s` を文字へ復号する方法の総数を返してください。

# 条件

- `"0"` 単体は復号できない
- `"10"` と `"20"` は復号できる
- `"06"` のように先頭が `0` の2桁は復号できない

# 例

### 例1

入力: `s = "12"`
出力: `2`
説明: `"AB"` と `"L"` の 2 通りです。

### 例2

入力: `s = "226"`
出力: `3`
説明: `"BZ"`, `"VF"`, `"BBF"` の 3 通りです。

### 例3

入力: `s = "06"`
出力: `0`

# 入力制約

- `1 <= s.length <= 100`
- `s` は数字のみで構成される
