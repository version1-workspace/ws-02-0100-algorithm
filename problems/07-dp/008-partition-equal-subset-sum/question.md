# 問題文

正の整数配列 `nums` が与えられます。
配列を2つの部分集合に分け、それぞれの合計を等しくできるかどうかを返してください。

# 条件

- 各要素はどちらか一方の部分集合に入れる
- 2つの部分集合の合計が同じになる必要がある
- 合計が奇数の場合は分割できない

# 例

### 例1

入力: `nums = [1,5,11,5]`
出力: `true`
説明: `[1,5,5]` と `[11]` に分けられます。

### 例2

入力: `nums = [1,2,3,5]`
出力: `false`

# 入力制約

- `1 <= nums.length <= 200`
- `1 <= nums[i] <= 100`
