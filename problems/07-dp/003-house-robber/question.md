title: House Robber

# 問題文

整数配列 `nums` が与えられます。
`nums[i]` は `i` 番目の家にある金額を表します。

隣り合う家を同じ夜に盗むと警報が鳴るため、隣接する家を同時に選ぶことはできません。
盗める金額の最大値を返してください。

# 条件

- 各家は盗むか盗まないかを選べる
- 隣り合う家を両方盗むことはできない
- 家が1軒だけの場合は、その家の金額を返す

# 例

### 例1

入力: `nums = [1,2,3,1]`
出力: `4`
説明: 1 番目と 3 番目の家を盗むと `1 + 3 = 4` です。

### 例2

入力: `nums = [2,7,9,3,1]`
出力: `12`
説明: `2 + 9 + 1 = 12` が最大です。

# 入力制約

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`
