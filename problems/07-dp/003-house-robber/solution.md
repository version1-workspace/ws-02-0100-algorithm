# 解説

## 考え方

各家について「盗む場合」と「盗まない場合」の最大金額を更新します。
今の家を盗むなら直前の家は盗めないので、直前までで盗まなかった場合に今の金額を足します。

## アルゴリズム

1. `skipPrevious` と `takePrevious` を 0 で初期化する
2. 各家の金額 `amount` を順に見る
3. 今の家を盗む場合は `skipPrevious + amount`
4. 今の家を盗まない場合は `max(skipPrevious, takePrevious)`
5. 最後に2状態の大きい方を返す

## 実装のポイント

- `takePrevious` は直前の家を盗んだ場合の最大金額です。
- `skipPrevious` は直前の家を盗まなかった場合の最大金額です。
- 配列ではなく2つの状態だけで十分です。

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(1)
