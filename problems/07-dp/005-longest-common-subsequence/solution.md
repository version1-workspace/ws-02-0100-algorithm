# 解説

## 考え方

`dp[i][j]` を `text1` の先頭 `i` 文字と `text2` の先頭 `j` 文字で作れる最長共通部分列の長さとします。
末尾の文字が同じなら両方使い、違うならどちらか一方の末尾を使わない場合の大きい方を選びます。

## アルゴリズム

1. `(text1.length + 1) x (text2.length + 1)` の `dp` を 0 で初期化する
2. `i` と `j` を 1 から順に進める
3. `text1[i - 1] === text2[j - 1]` なら `dp[i - 1][j - 1] + 1`
4. 違うなら `max(dp[i - 1][j], dp[i][j - 1])`
5. `dp[text1.length][text2.length]` を返す

## 実装のポイント

- 0行目と0列目は、片方が空文字のときの答えとして 0 にしておきます。
- `i` と `j` は `dp` 上の長さなので、文字列の添字は `i - 1`、`j - 1` になります。

## 計算量

- 時間計算量: O(text1.length * text2.length)
- 空間計算量: O(text1.length * text2.length)
