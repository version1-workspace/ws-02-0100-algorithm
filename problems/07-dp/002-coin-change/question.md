# 問題文

異なる額面の硬貨を表す配列 `coins` と、目標金額 `amount` が与えられます。
合計金額をちょうど `amount` にするために必要な硬貨の最小枚数を返してください。
作ることができない場合は `-1` を返してください。

# 条件

- 各額面の硬貨は何枚でも使える
- 硬貨の順番は結果に影響しない
- `amount = 0` の場合、必要な硬貨枚数は `0` とする

# 例

### 例1

入力: `coins = [1, 2, 5]`, `amount = 11`
出力: `3`
説明: `5 + 5 + 1 = 11` で 3 枚です。

### 例2

入力: `coins = [2]`, `amount = 3`
出力: `-1`

### 例3

入力: `coins = [1]`, `amount = 0`
出力: `0`

# 入力制約

- `1 <= coins.length <= 12`
- `1 <= coins[i] <= 2^31 - 1`
- `0 <= amount <= 10^4`
