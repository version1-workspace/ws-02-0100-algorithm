# 問題文

整数配列 `nums` が与えられます。
連続する部分配列の積のうち、最大値を返してください。

# 条件

- 部分配列は連続している必要がある
- 少なくとも1つの要素を選ぶ
- 負の値を掛けると最大値と最小値が入れ替わることに注意してください

# 例

### 例1

入力: `nums = [2,3,-2,4]`
出力: `6`
説明: `[2,3]` の積が最大です。

### 例2

入力: `nums = [-2,0,-1]`
出力: `0`

### 例3

入力: `nums = [-2,3,-4]`
出力: `24`

# 入力制約

- `1 <= nums.length <= 2 * 10^4`
- `-10 <= nums[i] <= 10`
- 答えは32bit整数の範囲に収まる
