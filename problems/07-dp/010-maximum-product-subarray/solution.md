# 解説

## 考え方

積では、負の数を掛けると最大値と最小値が入れ替わります。
そのため、各位置で終わる部分配列について最大積だけでなく最小積も持ちます。

## アルゴリズム

1. `maxEndingHere`、`minEndingHere`、`best` を `nums[0]` で初期化する
2. 2要素目以降を順に見る
3. 値が負なら、最大積と最小積を入れ替える
4. 今の値だけから始める場合と、前の積に掛ける場合の大きい方を最大積にする
5. 同様に小さい方を最小積にする
6. `best` を更新し、最後に返す

## 実装のポイント

- `maxEndingHere` は現在位置で終わる連続部分配列の最大積です。
- `minEndingHere` は負の値を掛けたときに最大になり得るため保持します。
- 0 が来た場合も `Math.max(value, maxEndingHere * value)` により、そこで再スタートできます。

## 計算量

- 時間計算量: O(n)
- 空間計算量: O(1)
