# 解説

## 考え方

各マスに到達する経路数は、上から来る経路数と左から来る経路数の合計です。
この実装では1行分の `dp` だけを持ち、左から右へ更新します。

## アルゴリズム

1. 列数 `n` の配列 `dp` をすべて 1 で初期化する
2. 2行目以降を順に処理する
3. 各列で `dp[column] += dp[column - 1]` と更新する
4. 右下に対応する `dp[n - 1]` を返す

## 実装のポイント

- 1行目と1列目は移動方法が1通りだけなので、初期値を 1 にできます。
- `dp[column]` は上から来る経路数、`dp[column - 1]` は左から来る経路数を表します。
- 行全体の表を持たず、1次元配列で十分です。

## 計算量

- 時間計算量: O(m * n)
- 空間計算量: O(n)
