# 問題文

`m x n` のグリッドがあります。
ロボットは左上のマスから開始し、右下のマスを目指します。

ロボットは1回の移動で右または下にだけ進めます。
右下のマスに到達する経路の総数を返してください。

# 条件

- 開始位置は左上
- 終了位置は右下
- 移動できる方向は右と下のみ

# 例

### 例1

入力: `m = 3, n = 7`
出力: `28`

### 例2

入力: `m = 3, n = 2`
出力: `3`
説明: 右下に行く方法は `3` 通りです。

# 入力制約

- `1 <= m, n <= 100`
- 答えは `2 * 10^9` 以下になる
