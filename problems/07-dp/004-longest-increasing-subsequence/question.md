---
title: Longest Increasing Subsequence
category: Dynamic Programming
---

# 問題文

整数配列 `nums` が与えられます。
配列からいくつかの要素を順番を保ったまま選び、値が厳密に増加する部分列を作ります。

作ることができる最長の増加部分列の長さを返してください。

# 条件

- 部分列は元の配列の順番を保つ
- 要素は連続している必要はない
- 同じ値は厳密な増加に含めない

# 例

### 例1

入力: `nums = [10,9,2,5,3,7,101,18]`
出力: `4`
説明: `[2,3,7,101]` などの長さ `4` の増加部分列があります。

### 例2

入力: `nums = [0,1,0,3,2,3]`
出力: `4`

# 入力制約

- `1 <= nums.length <= 2500`
- `-10^4 <= nums[i] <= 10^4`
