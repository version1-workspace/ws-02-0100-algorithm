# 解説

## 考え方

`dp[right]` を「`nums[right]` を最後の要素にする増加部分列の最大長」とします。
`right` より左にあり、かつ値が小さい要素 `left` だけが直前の要素になれます。

## アルゴリズム

1. すべての `dp` を 1 で初期化する
2. `right` を左から右へ動かす
3. `left < right` の全要素を調べる
4. `nums[left] < nums[right]` なら `dp[right] = max(dp[right], dp[left] + 1)` で更新する
5. 全体の最大値 `longest` を返す

## 実装のポイント

- 各要素単体でも長さ1の部分列なので、初期値は 1 です。
- 厳密な増加なので、比較は `nums[left] < nums[right]` です。
- この実装は O(n^2) の動的計画法です。

## 計算量

- 時間計算量: O(n^2)
- 空間計算量: O(n)
