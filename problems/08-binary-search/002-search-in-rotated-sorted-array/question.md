# 問題文

昇順にソートされた重複のない整数配列が、ある位置で回転された配列 `nums` と整数 `target` が与えられます。
`target` が `nums` に含まれている場合はそのインデックスを返してください。
含まれていない場合は `-1` を返してください。

例として `[0, 1, 2, 4, 5, 6, 7]` が回転されると、`[4, 5, 6, 7, 0, 1, 2]` のようになります。

# 条件

- `nums` の値は重複しない
- `nums` は昇順配列を回転したものである
- 計算量 `O(log n)` を目標とする

# 例

### 例1

入力: `nums = [4, 5, 6, 7, 0, 1, 2]`, `target = 0`
出力: `4`

### 例2

入力: `nums = [4, 5, 6, 7, 0, 1, 2]`, `target = 3`
出力: `-1`

### 例3

入力: `nums = [1]`, `target = 0`
出力: `-1`

# 入力制約

- `1 <= nums.length <= 5000`
- `-10^4 <= nums[i], target <= 10^4`
