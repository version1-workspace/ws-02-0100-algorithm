# 解説

## 考え方

配列は昇順にソートされているため、中央の値と `target` を比較すれば、探索範囲の半分を捨てられます。
これを範囲が空になるまで繰り返します。

## アルゴリズム

1. `left = 0`、`right = nums.length - 1` とする
2. `left <= right` の間、中央 `mid` を求める
3. `nums[mid]` が `target` なら `mid` を返す
4. `nums[mid] < target` なら左側を捨てて `left = mid + 1`
5. そうでなければ右側を捨てて `right = mid - 1`
6. 見つからなければ `-1` を返す

## 実装のポイント

- `left <= right` は閉区間 `[left, right]` を探索していることを表します。
- 空配列の場合は `right = -1` になり、ループに入らず `-1` を返します。

## 計算量

- 時間計算量: O(log n)
- 空間計算量: O(1)
