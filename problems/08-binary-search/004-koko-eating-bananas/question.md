# 問題文

バナナの山を表す整数配列 `piles` と、制限時間 `h` が与えられます。
ココは1時間に1つの山を選び、その山から最大 `k` 本のバナナを食べます。
山に残っているバナナが `k` 本未満なら、その山をすべて食べてその時間を終えます。

すべてのバナナを `h` 時間以内に食べ切るために必要な、最小の整数速度 `k` を返してください。

# 条件

- 1時間に食べられる山は1つだけである
- `k` は正の整数である
- `h` 時間以内に食べ切れる最小の `k` を求める
- 速度 `k` に対して必要時間を判定し、答えに対して二分探索する

# 例

### 例1

入力: `piles = [3, 6, 7, 11]`, `h = 8`
出力: `4`

### 例2

入力: `piles = [30, 11, 23, 4, 20]`, `h = 5`
出力: `30`

### 例3

入力: `piles = [30, 11, 23, 4, 20]`, `h = 6`
出力: `23`

# 入力制約

- `1 <= piles.length <= 10^4`
- `piles.length <= h <= 10^9`
- `1 <= piles[i] <= 10^9`
