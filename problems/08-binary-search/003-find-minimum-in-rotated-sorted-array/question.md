# 問題文

昇順にソートされた重複のない整数配列が、ある位置で回転された配列 `nums` が与えられます。
配列内の最小値を返してください。

例として `[0, 1, 2, 4, 5, 6, 7]` が回転されると、`[4, 5, 6, 7, 0, 1, 2]` のようになります。

# 条件

- `nums` の値は重複しない
- `nums` は昇順配列を回転したものである
- 回転されていない配列も入力される可能性がある
- 計算量 `O(log n)` を目標とする

# 例

### 例1

入力: `nums = [3, 4, 5, 1, 2]`
出力: `1`

### 例2

入力: `nums = [4, 5, 6, 7, 0, 1, 2]`
出力: `0`

### 例3

入力: `nums = [11, 13, 15, 17]`
出力: `11`

# 入力制約

- `1 <= nums.length <= 5000`
- `-5000 <= nums[i] <= 5000`
