# 解説

## 考え方

2 つのリストはすでに昇順なので、先頭同士を比べて小さい方を結果リストの末尾につなげていけば、昇順を保ったままマージできます。

## アルゴリズム

1. ダミーノード `dummy` と、末尾を指す `tail` を用意する
2. `current1` と `current2` で 2 つのリストを走査する
3. 両方が残っている間、値が小さい方のノードを `tail.next` につなぐ
4. つないだ側のポインタを次へ進め、`tail` も進める
5. 片方が空になったら、残りのリストを `tail.next` につなぐ
6. `dummy.next` を返す

## 実装のポイント

- `dummy` を使うことで、先頭ノードを特別扱いせずに済む
- 新しいノードを作らず、既存ノードの参照をつなぎ替えている
- `current1.val <= current2.val` の場合は `list1` 側を先につなぐ
- `tail.next = current1 ?? current2` で残りのリストをまとめて接続する

## 計算量

- 時間計算量: O(n + m)
- 空間計算量: O(1)
