# 問題文

昇順にソートされた2つの単方向連結リスト `list1` と `list2` が与えられます。
2つのリストを1つの昇順リストにマージし、マージ後の先頭ノードを返してください。

各ノードは次の `ListNode` で表されます。

```ts
class ListNode {
  val: number;
  next: ListNode | null;
}
```

# 条件

- 新しい値の配列を返すのではなく、マージ後の `ListNode | null` を返す
- どちらか一方のリストが空の場合は、もう一方のリストを返す
- マージ後のリストも昇順になるようにする

# 例

### 例1

入力: `list1 = [1, 2, 4], list2 = [1, 3, 4]`
出力: `[1, 1, 2, 3, 4, 4]`

### 例2

入力: `list1 = [], list2 = []`
出力: `[]`

### 例3

入力: `list1 = [], list2 = [0]`
出力: `[0]`

# 入力制約

- それぞれの連結リストのノード数は `0` 以上 `50` 以下
- `-100 <= Node.val <= 100`
- `list1` と `list2` は昇順にソートされている
