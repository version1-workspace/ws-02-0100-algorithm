# 問題文

単方向連結リストの先頭ノード `head` が与えられます。
連結リストの中に循環がある場合は `true`、循環がない場合は `false` を返してください。

循環とは、あるノードの `next` がリスト内の以前のノードを指している状態です。

各ノードは次の `ListNode` で表されます。

```ts
class ListNode {
  val: number;
  next: ListNode | null;
}
```

# 条件

- ノードの値ではなく、ノード同士の参照関係で循環を判定する
- 空の連結リストが与えられた場合は `false` を返す
- fast/slow pointer を使うと `O(1)` の追加メモリで判定できる

# 例

### 例1

入力: `head = [3, 2, 0, -4], pos = 1`
出力: `true`

説明: 末尾ノードがインデックス `1` のノードを指しているため循環があります。

### 例2

入力: `head = [1, 2], pos = 0`
出力: `true`

### 例3

入力: `head = [1], pos = -1`
出力: `false`

# 入力制約

- 連結リストのノード数は `0` 以上 `10000` 以下
- `-100000 <= Node.val <= 100000`
- `pos` は循環を作る位置を表す説明用の値であり、関数の引数には含まれない
