# 問題文

単方向連結リストの先頭ノード `head` が与えられます。
リストを `L0 -> Ln -> L1 -> Ln-1 -> L2 -> Ln-2 ...` の順に並べ替え、並べ替え後の先頭ノードを返してください。

各ノードは次の `ListNode` で表されます。

```ts
class ListNode {
  val: number;
  next: ListNode | null;
}
```

# 条件

- ノードの値は変更せず、`next` のつなぎ替えで並べ替える
- 空の連結リスト、またはノード数が1のリストはそのまま返す
- 中央探索、後半リストの反転、前半と後半のマージを組み合わせると効率よく解ける

# 例

### 例1

入力: `head = [1, 2, 3, 4]`
出力: `[1, 4, 2, 3]`

### 例2

入力: `head = [1, 2, 3, 4, 5]`
出力: `[1, 5, 2, 4, 3]`

### 例3

入力: `head = [1]`
出力: `[1]`

# 入力制約

- 連結リストのノード数は `0` 以上 `50000` 以下
- `1 <= Node.val <= 1000`
