# 問題文

単方向連結リストの先頭ノード `head` が与えられます。
連結リストの向きを反転し、反転後の先頭ノードを返してください。

各ノードは次の `ListNode` で表されます。

```ts
class ListNode {
  val: number;
  next: ListNode | null;
}
```

# 条件

- 新しい値の配列を返すのではなく、反転後の `ListNode | null` を返す
- 空の連結リストが与えられた場合は `null` を返す
- ノードの値は変更せず、`next` の向きだけを変更する

# 例

### 例1

入力: `head = [1, 2, 3, 4, 5]`
出力: `[5, 4, 3, 2, 1]`

説明: `1 -> 2 -> 3 -> 4 -> 5` を `5 -> 4 -> 3 -> 2 -> 1` に反転します。

### 例2

入力: `head = [1, 2]`
出力: `[2, 1]`

### 例3

入力: `head = []`
出力: `[]`

# 入力制約

- 連結リストのノード数は `0` 以上 `5000` 以下
- `-5000 <= Node.val <= 5000`
