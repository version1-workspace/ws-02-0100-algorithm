---
title: Remove Nth Node From End
category: Linked List
---

# 問題文

単方向連結リストの先頭ノード `head` と整数 `n` が与えられます。
リストの末尾から数えて `n` 番目のノードを削除し、削除後の先頭ノードを返してください。

各ノードは次の `ListNode` で表されます。

```ts
class ListNode {
  val: number;
  next: ListNode | null;
}
```

# 条件

- 新しい値の配列を返すのではなく、削除後の `ListNode | null` を返す
- `n` は必ず有効な範囲で与えられる
- 2つのポインタを使うと1回の走査で削除できる

# 例

### 例1

入力: `head = [1, 2, 3, 4, 5], n = 2`
出力: `[1, 2, 3, 5]`

### 例2

入力: `head = [1], n = 1`
出力: `[]`

### 例3

入力: `head = [1, 2], n = 1`
出力: `[1]`

# 入力制約

- 連結リストのノード数は `1` 以上 `30` 以下
- `1 <= n <= 連結リストのノード数`
- `0 <= Node.val <= 100`
