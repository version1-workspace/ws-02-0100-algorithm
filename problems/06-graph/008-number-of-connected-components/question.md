# 問題文

`n` 個のノードと無向辺の一覧 `edges` が与えられます。
ノードは `0` から `n - 1` までの番号で表されます。

この無向グラフに含まれる連結成分の数を返してください。

# 条件

- 辺で直接または間接的につながるノードは同じ連結成分に属する
- 辺を持たないノードは単独の連結成分として数える
- Union-Find または DFS で連結成分を数えられる

# 例

### 例1

入力: `n = 5`, `edges = [[0, 1], [1, 2], [3, 4]]`
出力: `2`

### 例2

入力: `n = 5`, `edges = [[0, 1], [1, 2], [2, 3], [3, 4]]`
出力: `1`

# 入力制約

- `1 <= n <= 2000`
- `0 <= edges.length <= 5000`
- `edges[i].length = 2`
- `0 <= edges[i][0], edges[i][1] < n`
