# 問題文

`n` 個のノードと無向辺の一覧 `edges` が与えられます。
ノードは `0` から `n - 1` までの番号で表されます。

この無向グラフが木である場合は `true`、そうでない場合は `false` を返してください。

# 条件

- 木はすべてのノードが連結で、循環を持たないグラフである
- `n` 個のノードからなる木の辺数は必ず `n - 1` 本である
- Union-Find または DFS で連結性と循環を判定できる

# 例

### 例1

入力: `n = 5`, `edges = [[0, 1], [0, 2], [0, 3], [1, 4]]`
出力: `true`

### 例2

入力: `n = 5`, `edges = [[0, 1], [1, 2], [2, 3], [1, 3], [1, 4]]`
出力: `false`

# 入力制約

- `1 <= n <= 2000`
- `0 <= edges.length <= 5000`
- `edges[i].length = 2`
- `0 <= edges[i][0], edges[i][1] < n`
