# 解説

## 考え方

腐敗はすべての腐ったオレンジから同時に 1 分ずつ広がります。これは複数の始点を持つ BFS として扱えます。

最初に腐ったオレンジをすべてキューに入れ、新鮮なオレンジの数を数えます。BFS で隣接する新鮮なオレンジを腐らせながら、何分目に腐ったかを記録します。

## アルゴリズム

1. グリッドを走査し、新鮮なオレンジ数 `freshCount` を数える
2. 腐ったオレンジを `[row, col, 0]` として `queue` に入れる
3. `queue` を先頭から順に処理する
4. 現在セルの上下左右にある新鮮なオレンジを腐らせる
5. 腐らせたセルを `[nextRow, nextCol, currentMinute + 1]` として `queue` に追加する
6. BFS 後、`freshCount` が 0 なら経過時間 `minutes`、残っていれば `-1` を返す

## 実装のポイント

- `queue` には座標に加えて、そのオレンジが腐った時刻を入れています。
- 新鮮なオレンジを腐らせた時点で `grid[nextRow][nextCol] = 2` に更新し、同じセルを重複してキューに入れないようにします。
- `minutes` は処理したセルの `currentMinute` の最大値です。
- 最初から新鮮なオレンジがない場合、BFS 後も `freshCount === 0` なので `0` が返ります。

## 計算量

- 時間計算量: O(RC)
- 空間計算量: O(RC)

ただし `R` は行数、`C` は列数です。各セルは高々一度キューに入ります。
