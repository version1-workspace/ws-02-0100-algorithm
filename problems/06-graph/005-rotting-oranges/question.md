# 問題文

オレンジの状態を表す 2 次元グリッド `grid` が与えられます。

- `0` は空のセル
- `1` は新鮮なオレンジ
- `2` は腐ったオレンジ

腐ったオレンジは1分ごとに、上下左右に隣接する新鮮なオレンジを腐らせます。
すべての新鮮なオレンジが腐るまでに必要な最短時間を返してください。
すべて腐らせることができない場合は `-1` を返してください。

# 条件

- 斜め方向には腐敗は広がらない
- 最初から新鮮なオレンジがない場合は `0` を返す
- 複数の腐ったオレンジを始点にした BFS として考える

# 例

### 例1

入力: `grid = [[2, 1, 1], [1, 1, 0], [0, 1, 1]]`
出力: `4`

### 例2

入力: `grid = [[2, 1, 1], [0, 1, 1], [1, 0, 1]]`
出力: `-1`

### 例3

入力: `grid = [[0, 2]]`
出力: `0`

# 入力制約

- `1 <= grid.length <= 10`
- `1 <= grid[i].length <= 10`
- `grid[i][j]` は `0`, `1`, `2` のいずれか
