# 問題文

無向連結グラフが隣接リスト `adjList` として与えられます。
元のグラフと同じ構造を持つ新しいグラフを作り、その隣接リスト表現を返してください。

この問題のテスト用インターフェースでは、`adjList[i]` は値 `i + 1` のノードに隣接するノードの値一覧を表します。
返り値も同じ `number[][]` 形式にしてください。

# 条件

- 元のグラフの参照を使い回さず、新しいノードで複製する
- 無向グラフとして扱う
- 空のグラフが与えられた場合は `[]` を返す
- DFS または BFS と HashMap を使うと、対応する複製ノードを管理できる

# 例

### 例1

入力: `adjList = [[2, 4], [1, 3], [2, 4], [1, 3]]`
出力: `[[2, 4], [1, 3], [2, 4], [1, 3]]`

### 例2

入力: `adjList = [[]]`
出力: `[[]]`

### 例3

入力: `adjList = []`
出力: `[]`

# 入力制約

- `0 <= adjList.length <= 100`
- ノードの値は `1` から `adjList.length` までの整数
- `adjList[i]` の各値は隣接ノードの値を表す
