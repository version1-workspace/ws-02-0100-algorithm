# 解説

## 考え方

島は上下左右につながった `"1"` のまとまりです。まだ訪れていない陸地を見つけたら新しい島として数え、その島に属する陸地を DFS ですべて訪問済みにします。

一度訪問した陸地は同じ島として処理済みなので、後から数え直さないようにします。

## アルゴリズム

1. `visited` を用意し、各セルを未訪問として初期化する
2. グリッド全体を左上から順に走査する
3. セルが `"1"` かつ未訪問なら、島の数 `count` を 1 増やす
4. `visitIsland` でそのセルから上下左右に DFS し、同じ島の陸地を訪問済みにする
5. 走査が終わったら `count` を返す

## 実装のポイント

- `visited[row][col]` は、そのセルをすでにどこかの島として処理したかを表します。
- `visitIsland` は再帰ではなく `stack` を使った DFS です。最大 300 x 300 のグリッドでも、再帰の深さを気にせず処理できます。
- 範囲外、水セル、訪問済みセルは `continue` で無視します。
- 問題の制約では `grid.length >= 1` ですが、実装には空配列への防御として `0` を返す処理があります。

## 計算量

- 時間計算量: O(RC)
- 空間計算量: O(RC)

ただし `R` は行数、`C` は列数です。各セルは高々一度だけ訪問済みとして処理されます。
