# 問題文

各セルの高さを表す `heights` が与えられます。
太平洋は上端と左端に、大西洋は下端と右端に接しています。

水は現在のセルから、上下左右に隣接する同じ高さ以下のセルへ流れます。
太平洋と大西洋の両方へ水が流れ着けるセルの座標を返してください。

runner で比較できるよう、返り値は `[row, col]` の配列を行優先の昇順で返してください。

# 条件

- 斜め方向には移動できない
- 各セルから海へ直接シミュレーションしてもよいが、海側から逆向きにたどると効率よく判定できる
- 逆向きにたどる場合は、現在のセル以上の高さの隣接セルへ進む

# 例

### 例1

入力:

```text
heights = [
  [1, 2, 2, 3, 5],
  [3, 2, 3, 4, 4],
  [2, 4, 5, 3, 1],
  [6, 7, 1, 4, 5],
  [5, 1, 1, 2, 4]
]
```

出力: `[[0, 4], [1, 3], [1, 4], [2, 2], [3, 0], [3, 1], [4, 0]]`

### 例2

入力: `heights = [[1]]`
出力: `[[0, 0]]`

# 入力制約

- `1 <= heights.length <= 200`
- `1 <= heights[i].length <= 200`
- `0 <= heights[i][j] <= 100000`
