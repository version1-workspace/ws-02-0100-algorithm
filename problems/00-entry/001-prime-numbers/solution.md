# 解説

## 考え方

素数は `1` より大きく、`1` と自分自身以外で割り切れない整数です。`n` が合成数なら、必ず `sqrt(n)` 以下の約数を持つため、すべての数を試す必要はありません。

## アルゴリズム

1. `n <= 1` なら素数ではないので `false` を返す
2. `n === 2` なら素数なので `true` を返す
3. `2` 以外の偶数なら `false` を返す
4. `3` から `sqrt(n)` まで奇数だけを試し、割り切れたら `false` を返す
5. 最後まで割り切れなければ `true` を返す

## 実装のポイント

- `divisor * divisor <= n` とすることで、平方根を直接計算せずに探索範囲を制限している
- 偶数を先に除外するため、ループでは `divisor += 2` として奇数だけを調べる
- `n <= 1`、`n === 2`、偶数を先に処理して、境界条件を明確にしている

## 計算量

- 時間計算量: O(sqrt(n))
- 空間計算量: O(1)
