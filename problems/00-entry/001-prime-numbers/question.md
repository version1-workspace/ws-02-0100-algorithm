# 問題文

整数 `n` が与えられます。
`n` が素数であれば `true`、素数でなければ `false` を返してください。

素数とは、`1` より大きい整数で、`1` と自分自身以外に正の約数を持たない数です。

# 条件

- `1` 以下の整数は素数ではない
- `2` は素数である
- 偶数で素数なのは `2` のみである

# 例

### 例1

入力: `n = 2`
出力: `true`

### 例2

入力: `n = 9`
出力: `false`
説明: `9` は `3` で割り切れるため素数ではありません。

### 例3

入力: `n = 1`
出力: `false`

# 入力制約

- `-10^9 <= n <= 10^9`
